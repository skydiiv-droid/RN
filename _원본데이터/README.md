# _원본데이터 (raw)

data.go.kr에서 내려받은 **원본 파일**을 여기에 올립니다. (CSV / 엑셀 / ZIP)

Claude가 이 폴더의 파일을 읽어 `약학/` 아래 노트로 변환합니다.
변환이 끝나면 원본은 남겨둬도 되고, 용량이 크면 지워도 됩니다(노트만 있으면 됨).

## 올리는 법 (폰/PC 공통)

1. GitHub에서 이 저장소 → `_원본데이터` 폴더로 이동
2. **Add file → Upload files** (폰은 브라우저 데스크톱 모드에서 가능)
3. 내려받은 파일 선택 → Commit

## 주의

- **한 파일 100MB 초과 불가** (GitHub 제한). 크면 알려주세요 — 자주 쓰는 약만 걸러서 처리합니다.
- 파일에 **환자 정보가 들어갈 일은 없음** (식약처 공개 데이터). 혹시라도 개인정보가 섞인 파일은 올리지 마세요.
