# 제품 (e약은요)

**제품(품목) 단위** 노트. 식약처 e약은요(의약품개요정보) 기준. 제품 하나당 노트 하나.

## 각 노트에 담기는 것

- 효능 / 용법·용량 / 사용 전 주의 / 사용상 주의사항 / 함께 주의할 약·음식 / 이상반응(부작용) / 보관법
- 매칭되는 경우 **생김새(낱알식별)** — 성상·색·제형·표시·이미지

## 속성으로 검색·분류

```yaml
구분: 일반의약품     # 일반의약품 / 전문의약품 / 미상
효능군: "해열.진통.소염제"
```

전문/일반 구분은 낱알식별 데이터가 있는 제품만 채워지고, 없으면 `미상`입니다.
폴더를 나누지 않고 `구분` 속성으로 거르는 것이 이 저장소 원칙(얕은 폴더 + 속성)에 맞습니다.

## 참고

- 원본 텍스트는 e약은요 특성상 **띄어쓰기가 없습니다**(원본 보존). 숫자 오분리 복구와 문장 단위 줄바꿈만 적용했습니다.
- 성분단위 DUR(병용금기 등)은 `약학/성분/` 노트를 참고. (현재 제품↔성분 자동 링크는 없음 — 향후 허가정보로 연결 예정)

> ℹ️ 참고용. 최종 판단은 처방·원내 지침·최신 허가사항을 따릅니다.
