# 인터큐텐연질캡슐

> ℹ️ 식약처 e약은요(의약품개요정보) · 참고용. 최종 판단은 처방·원내 지침을 따릅니다.

## 효능
- 이약은육체피로
- 임신ㆍ수유기
- 병중ㆍ병후(병을앓는동안이나회복후)의체력저하시
- 노년기의비타민A
- C
- B1
- E의보급과눈의건조감의완화
- 야맹증(밤에잘못보는증상)
- 햇빛ㆍ피부병등에의한색소침착(기미
- 주근깨)의완화
- 잇몸출혈ㆍ비출혈(코피)예방
- 말초혈행장애및갱년기시어깨ㆍ목결림
- 수족저림ㆍ수족냉증(손발이차가움)증상의완화에사용합니다.

## 용법·용량
- 만12세이상및성인은 1회1캡슐씩
- 1일1회복용합니다.

## 사용 전 반드시 알아야 할 내용
- 비타민A를1일5,000IU이상복용시선천성기형을일으킬수있습니다.
- 임신3개월이내또는임신하고있을가능성이있는여성은복용하지마십시오.

- 국한성대장염등의위장질환
