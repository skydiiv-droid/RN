# 구미히비크린에이액(글루콘산클로르헥시딘액)

## 효능
- 이약은항균피부세정제로서
- 일반피부및상처부위세정
- 수술시손소독
- 개인의손소독
- 수술전수술부위피부소독에사용합니다.

## 용법·용량
- 일반피부및상처부위세정에는세정할부위를물로닦고
- 피부나상처에적용하는데필요한최소량만을덜어적용한뒤
- 부드럽게닦아내십시오.물로충분히헹굽니다.
- 수술시의손소독에는손과팔뚝을물로적시고본제품5mL를손과팔꿈치까지적용하고약3분간솔또는스폰지를사용하여문지릅니다.
- 흐르는물에충분히헹구고위과정을한번더반복한후완전히말립니다.
- 개인의손소독에는손을물로적시고본제품5mL를손에바른뒤양손을15초간강하게문지릅니다.
- 물로헹구고완전히말립니다.
- 수술전피부소독에는충분한양만큼수술부위에적용하고
- 2분이상문지릅니다.
- 멸균수건으로말리고위과정을한번더반복한후멸균수건으로말립니다.

- 임부또는임신하고있을가능성이있는부인또는수유부는 의사또는약사와상의하십시오.이약은외용으로만사용하고경구투여및주사하지마십시오.눈에접촉시각막손상을일으킬수있으므로눈에들어가지않도록주의하고만약들어갔을경우에는즉시깨끗한물로잘씻어내십시오.이약은섬유및헝겊(면
- 거즈
- 울
- 레이온등)에흡착되므로용액에적셔서사용시유효농도이상이아닐경우새로운용액을공급하십시오.식염수등에들어있는음이온에의하여난용성염을생성하므로희석하는경우에는깨끗한증류수를사용하십시오.손의세정등에사용하는희석액은적어도매일새로운용액으로교환하십시오.
