# 이지페인록센연질캡슐(나프록센)

> ℹ️ 식약처 e약은요(의약품개요정보) · 참고용. 최종 판단은 처방·원내 지침을 따릅니다.

## 효능
- 이약은류마티양관절염
- 골관절염(퇴행성관절염)
- 강직성척추염
- 건염(힘줄염)
- 급성통풍
- 월경곤란증에사용합니다.
- 이약은삠
- 타박상
- 상처
- 허리통증등의골격근장애
- 활액낭염
- 수술후통증
- 편두통
- 발치후통증에사용합니다.

## 용법·용량
- 류마티양관절염
- 강직성척추염에는성인은1회1~2캡슐씩
- 1일2회(12시간마다)복용합니다.
- 급성통풍에는성인은초회량으로3캡슐을복용하고
- 발작이소실될때까지8시간간격으로1캡슐을복용합니다.
- 골격근장애
- 수술후통증
- 발치후통증
- 월경곤란증
- 힘줄염
- 활액낭염에는성인은초회량으로2캡슐을복용한후
- 6~8시간간격으로1캡슐씩복용합니다.
- 편두통에는성인은초회량으로3캡슐을복용합니다.
- 필요시1일1~2캡슐을더복용할수있으며
- 초회량복용30분후에복용합니다.
- 1일총용량이5캡슐을초과하지마십시오.나이
- 증상에따라적절히증감하십시오.

## 사용 전 반드시 알아야 할 내용
- 매일세잔이상정기적음주자가이약또는다른해열진통제를복용할때는의사또는약사와상의하십시오.위장출혈을일으킬수있습니다.
- 이약복용시치명적일수있는중대한심혈관계혈전반응
- 심근경색증
- 뇌졸중의위험이증가하며
- 위장관출혈
- 궤양및천공등심각한위장관계이상반응이나타날수있습니다.
