# 약학

국내 사용 의약품 정보. 데이터 출처는 **식품의약품안전처 공공데이터**(공공데이터포털 data.go.kr)입니다.

- `_data/` — **제품(품목) 데이터베이스** 35,206개 (허가정보+e약은요+낱알식별 병합). 노트가 아니라 데이터 파일 — 검색 도구에서 사용.
- `성분/` — 성분별 DUR 안전정보 노트 (병용금기·임부금기·연령금기·노인주의). 서로 링크되는 진짜 노트라 Obsidian에서 관리.

> 제품은 3.5만 개라 "노트"가 아닌 "데이터"로 둡니다(폰 성능·용량). 성분 DUR만 노트입니다.

## 노트 = 약 하나

약 하나당 노트 하나. 파일명은 제품명 그대로 (예: `타이레놀정500mg.md`).
새 노트는 [[_템플릿/약품노트]] 를 복사해서 시작합니다.

## 태그/속성 예시

```yaml
---
성분: 아세트아미노펜
효능군: [해열제, 진통제]
구분: 일반의약품
품목기준코드: "195700020"
---
```

## DUR(병용금기 등)은 위키링크로

병용금기·중복 등 약끼리의 관계는 `[[와파린]]` 처럼 링크로 걸어 둡니다.
그러면 Obsidian 그래프 뷰에서 "이 약과 엮인 금기 약들"이 시각적으로 연결됩니다.
