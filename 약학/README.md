# 약학

국내 사용 의약품 정보. 데이터 출처는 **식품의약품안전처 공공데이터**(공공데이터포털 data.go.kr)입니다.

- `제품/` — 제품(품목)별 노트 (e약은요: 효능·용법·부작용·생김새). 전문/일반은 `구분` 속성으로 구분.
- `성분/` — 성분별 DUR 안전정보 (병용금기·임부금기·연령금기·노인주의)
- `일반의약품/`, `전문의약품/` — (예비 폴더. 실제 제품 노트는 `제품/`에 통합, 구분은 속성으로)

## 노트 = 약 하나

약 하나당 노트 하나. 파일명은 제품명 그대로 (예: `타이레놀정500mg.md`).
새 노트는 [[_템플릿/약품노트]] 를 복사해서 시작합니다.

## 태그/속성 예시

```yaml
---
성분: 아세트아미노펜
효능군: [해열제, 진통제]
구분: 일반의약품
품목기준코드: "195700020"
---
```

## DUR(병용금기 등)은 위키링크로

병용금기·중복 등 약끼리의 관계는 `[[와파린]]` 처럼 링크로 걸어 둡니다.
그러면 Obsidian 그래프 뷰에서 "이 약과 엮인 금기 약들"이 시각적으로 연결됩니다.
