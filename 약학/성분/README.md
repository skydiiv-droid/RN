# 성분 (DUR 안전정보)

**성분(成分) 단위** DUR 안전정보 노트. 성분 하나당 노트 하나.

DUR 데이터는 제품 단위(수십만 행)라 그대로 쓰면 너무 크고 중복됩니다.
임상 규칙은 성분 쌍 단위이므로, **성분명 기준으로 압축**해 정리합니다.

## 각 노트에 담기는 것

- 🚫 **병용금기** — 함께 쓰면 안 되는 다른 성분 (`[[성분]]` 링크 → 그래프로 연결)
- 🤰 **임부금기** — 등급 + 사유
- 👶 **연령금기** — 연령 조건 + 사유
- 👴 **노인주의** — 주의 내용

## 속성으로 검색

```yaml
임부금기: true
병용금기수: 24
```

Obsidian에서 `임부금기: true` 등으로 필터하면 해당 성분이 한 번에 모입니다.
