# 성분 (DUR 안전정보)

**성분(成分) 단위** DUR 안전정보 노트. 성분 하나당 노트 하나. (한글 성분명 기준)

출처: **식품의약품안전처 DUR 성분정보**(2026.7) — 병원 DUR 시스템의 공식 원본 데이터.

## 담기는 DUR 8종

- 🚫 **병용금기** — 함께 쓰면 안 되는 성분 (`[[성분]]` 링크, 양방향)
- 🤰 **임부금기** — 등급 + 사유
- 👶 **특정연령대금기** — 연령기준 + 사유
- 👴 **노인주의**
- ⚖️ **용량주의** — 1일 최대용량
- ⏱️ **투여기간주의** — 최대 투여기간
- 🔁 **효능군중복주의**
- 🧪 **첨가제주의**

## 속성으로 검색

```yaml
병용금기: true
임부금기: true
병용금기수: 3
```

Obsidian에서 `임부금기: true` 등으로 필터하면 해당 성분이 한 번에 모입니다.
병용금기는 성분 간 링크라 그래프 뷰에서 관계망으로 보입니다.

> ℹ️ 참고용. 최종 판단은 처방·원내 지침을 따릅니다.
