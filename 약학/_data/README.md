# _data (의약품 데이터베이스)

제품(품목) 정보는 **노트가 아니라 데이터**로 관리합니다. 3.5만 개를 노트 파일로 만들면 무겁고, 이건 "검색해서 찾는" 데이터베이스 성격이기 때문입니다.

## 파일

- **의약품.ndjson** — 정상 유통 의약품 **35,206개** 전체 상세 (1줄 = 1제품, JSON)
- **의약품_색인.json** — 검색용 경량 색인 (id·제품명·성분·구분·효능군)

## 출처(병합)

품목일련번호 기준으로 3개 데이터를 합쳤습니다:
- **허가정보**(마스터): 전문/일반·성분·효능효과·용법용량·주의사항·ATC·마약류
- **e약은요**: 환자용 쉬운 효능·용법·부작용·상호작용·보관 (4,746개)
- **낱알식별**: 성상·색·제형·표시·이미지 (21,401개)

## 레코드 예시

```json
{"id":"197900277","명":"게보린정","구분":"일반의약품","성분":{"주성분":"...","영문":"..."},
 "효능효과":"...","용법용량":"...","주의사항":"...","마약류":"","ATC":"...",
 "e약은요":{"효능":"...","용법":"...","부작용":"..."},"생김새":{"색상앞":"분홍","이미지":"http..."}}
```

## 사용

이 데이터는 **검색 도구**(별도 예정)에서 읽어 약 이름·성분으로 검색합니다.
성분 단위 DUR(병용금기 등)은 `약학/성분/` 노트를 참고.

> ℹ️ 참고용. 최종 판단은 처방·원내 지침·최신 허가사항을 따릅니다.
> 데이터 기준일: 2026-07 (식약처·심평원 공공데이터)
