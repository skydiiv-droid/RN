# Pulmonary Function Test (폐기능검사)

> ℹ️ 참고용 · 판독은 의사. 준비·간호는 원내 지침.

## 용도
폐활량·기류제한 평가. 천식·COPD 진단·중증도.

## 준비·과정
- 기관지확장제 보류 지침
- 최대 노력 협조 필요

## ⚠️ 주의
- 최근 심근경색·기흉 등 금기 확인

## 간호
- 검사 협조·호흡법 교육
