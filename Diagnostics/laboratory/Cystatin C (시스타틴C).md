# Cystatin C (시스타틴C)

> ℹ️ 참고용 · 정상범위는 검사실·방법마다 다름. **최종은 우리 검사실 참고치 확인.**

## 정상범위
- **0.61–0.95 mg/L mg/L** — 근육량·나이 영향이 적어 크레아티닌보다 정밀한 신기능 지표

## ↑ 상승
- 신기능 저하(eGFR↓)
- 갑상선기능항진·스테로이드

## ↓ 저하
- 갑상선기능저하

## 간호·임상 포인트
- 크레아티닌이 부정확한 경우(고령·근감소·소아) 유용
- 조기 신기능 저하 민감
