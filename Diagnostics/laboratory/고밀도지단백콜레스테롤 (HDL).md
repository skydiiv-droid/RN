# 고밀도지단백콜레스테롤 (HDL, HDL Cholesterol)

> ℹ️ 참고용 · 정상범위는 검사실·장비·방법마다 다름. **최종은 우리 검사실 참고치 확인.**

- **남 >40 / 여 >50 mg/dL** — "좋은" 콜레스테롤 — 높을수록 보호적
