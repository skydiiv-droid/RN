# C형간염 표지자 (anti-HCV / HCV RNA)

> ℹ️ 참고용 · 최종 판단은 임상·전문의.

## 표지자
- **anti-HCV (항체)** — 노출/감염(과거 or 현재). 양성 → 확인검사 필요
- **HCV RNA (PCR)** — **현재 감염 확진**·치료 지표. 항체(+)+RNA(+) = 현증 감염
- 항체(+) + RNA(−) → 과거 감염·회복(항체는 방어면역 아님)

## 간호 포인트
- **혈액 매개** → 표준주의·바늘자상 예방 (B형과 달리 백신 없음)
- DAA 항바이러스제로 완치율 높음 → 치료 연계
