# 전혈구계산 (CBC, Complete Blood Count)

> ℹ️ 참고용 · 정상범위는 검사실·장비·연령·성별마다 다름. **최종은 우리 검사실 참고치 확인.**

혈액의 세 세포계열(백혈구·적혈구·혈소판)을 한 번에 보는 기본 검사.

## 구성별 정상범위 · 의미

| 항목 | 정상범위(성인) | ↑ | ↓ |
|---|---|---|---|
| **WBC** 백혈구 | 4,000–10,000 /μL | 감염·염증·백혈병·스테로이드 | 골수억제·[[항암제]]·바이러스·패혈증 |
| **Hb** 혈색소 | 남 13–17 / 여 12–16 g/dL | 적혈구증가·탈수 | **[[빈혈]]**·출혈 |
| **Hct** 적혈구용적 | 남 40–52 / 여 36–48 % | 탈수·다혈증 | 빈혈·수분과다 |
| **PLT** 혈소판 | 150,000–400,000 /μL | 혈전·염증·반응성 | **출혈위험**·[[혈소판감소증]] |
| RBC 적혈구 | 남 4.5–5.9 / 여 4.0–5.2 ×10⁶/μL | 다혈증 | 빈혈 |

## 백혈구 감별계산 (WBC differential)
- **호중구(Neutrophil)** 40–75% — ↑세균감염·염증 / ↓골수억제
- **림프구(Lymphocyte)** 20–45% — ↑바이러스 / ↓면역저하·스테로이드
- 단핵구·호산구·호염기구
- **ANC(절대호중구수)**: 감염위험 지표 — **<500 = 중증 호중구감소(감염 고위험)** → 역격리 고려

## 적혈구·혈소판 지수
- **MCV** 80–98 fL — 적혈구 크기 → ↓소구성(철결핍) / ↑대구성(B12·엽산 결핍)
- **MCH** 27–33 pg · **MCHC** 32–35 g/dL — 혈색소 함량 → ↓저색소성(철결핍)
- **RDW** 11.5–14.5 % — 크기 불균일 → ↑철결핍·혼합빈혈
- **MPV / PDW / PCT** — 혈소판 크기·분포·용적(생성 활성 참고)
- **MCV + RDW 조합**으로 [[빈혈]] 감별에 유용

## 간호 포인트
- **WBC↑ + 발열** → 감염 사정, 배양 확인
- **PLT↓ (<50,000)** → 출혈 징후(점상출혈·잇몸출혈), 침습적 처치·근육주사 주의
- **ANC↓** → 감염 예방(손위생·역격리), 활력징후·발열 밀착 감시
- **Hb↓** → 빈혈 증상(어지럼·창백·빈맥), 수혈 여부 확인
