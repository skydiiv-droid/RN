# 혈액형·항체선별 (ABO/Rh Typing, Antibody Screening)

> ℹ️ 참고용 · 수혈 관련은 원내 수혈지침·혈액은행 절차 최우선.

## 구성
- **ABO cell/serum typing** — 적혈구 항원(cell) + 혈청 항체(serum) 교차확인 → A/B/O/AB
- **Rh(D) typing** — Rh 양성(+)/음성(−)
- **비예기항체 선별(Ab screening)** — 수혈·임신으로 생긴 예기치 못한 항체 유무
  - **Negative** = 적합혈 찾기 쉬움 / **Positive** → 항체 동정 필요, 적합혈 제한

## 간호 포인트 ⭐ (수혈 안전)
- 수혈 전 **혈액형·교차시험(crossmatch)·환자확인(2인)** 필수
- **수혈 부작용**(발열·오한·발진·호흡곤란·요통) 발생 시 즉시 중단·보고
- Rh(−) 임부: 항D면역글로불린 관련 확인
