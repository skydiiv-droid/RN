# HBV markers (B형간염표지자)

> ℹ️ 참고용 · 최종 판단은 임상·전문의.

## 주요 표지자

| 표지자 | 양성 의미 |
|---|---|
| **HBsAg** (표면항원) | **현재 감염**(급성/만성) |
| **anti-HBs** (표면항체) | **면역**(회복 or 백신) |
| **HBeAg** (e항원) | 활발한 증식·**높은 전염력** |
| **anti-HBe** (e항체) | 증식 감소·전염력↓ |
| **anti-HBc IgM** | **급성** 감염 |
| **anti-HBc IgG** | 과거/만성 노출 |
| **HBV DNA** | 바이러스 양(치료 지표) |

## 해석 패턴

- **HBsAg(+), anti-HBc(+), anti-HBs(−)** → 감염(급성/만성)
- **HBsAg(−), anti-HBs(+), anti-HBc(+)** → 자연 회복·면역
- **HBsAg(−), anti-HBs(+), anti-HBc(−)** → **백신 접종 면역**
- **모두 음성** → 미감염(백신 대상)

## 간호 포인트
- **혈액·체액 매개** → 표준주의·바늘자상 예방
- HBsAg 양성 환자 처치 후 노출 시 즉시 보고(PEP)
- 백신 미접종·미면역 의료인 접종 권장
